# TV Redirector

Веб-сервис для управления редиректом на ТВ-листы. Администратор задаёт ссылку через панель управления, а пользователи переходят по короткому адресу `/tv` и автоматически перенаправляются на актуальную трансляцию.

## Стек

- **FastAPI** + **Uvicorn** — бэкенд
- **SQLAlchemy** + **SQLite** — хранение данных
- **Jinja2** + **Bootstrap 5** — шаблоны админ-панели
- **Nginx** — реверс-прокси с SSL
- **Docker Compose** — оркестрация

## Быстрый старт

### 1. Создайте файл `.env` в корне проекта

```
ADMIN_USERNAME=admin
ADMIN_PASSWORD=ваш_пароль
```

### 2. Запустите через Docker Compose

```bash
docker compose up -d --build
```

Приложение будет доступно на порту `8000` (напрямую) или через Nginx на портах `80`/`443`.

## Маршруты

| Метод | Путь             | Описание                          |
|-------|------------------|-----------------------------------|
| GET   | `/tv`            | Редирект на актуальную ссылку     |
| GET   | `/admin`         | Панель администратора / логин     |
| POST  | `/admin/login`   | Авторизация                       |
| POST  | `/admin/update`  | Обновление ссылки на трансляцию   |

## Локальный запуск (без Docker)

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cd app
uvicorn main:app --host 0.0.0.0 --port 8000
```
